A Helm Chart collection

Popular applications, provided by Jon Zuka, ready to launch on Kubernetes using Kubernetes Helm.

TL;DR

$ helm repo add jonzuka https://jon-zuka.github.io/helm-chart
$ helm search repo <chart> --devel
$ helm install my-release jonzuka/<chart>

For more information, please refer to Jon Zuka's Helm Chart collection on GitHub.
